What are the best practices associated with using classes vs. ids?

Ids

Ids are unique identifiers. You use the # to identify id's in CSS file.




Once you use it once you cant use it again in the same document. You cant have two ids called main.



Think of an id like your drivers license number. Your drivers license number helps to identify you and you cant have two drivers licenses for different people with the same number. Therefore you use IDs when you want to select something specific.

Classes

Classes are great classifiers. They use the . (dot) to identify the class in the CSS.



They are not unique in nature. You can also only have one id more element but you can have multiple classes per element.



Classes are used when you want to apply the same styling to multiple elements.



Id gets higher priority than class in css. Meaning that any styling in css for the id will override the styling that has been given to the class.
