What are the best practices associated with using classes vs. ids?

Id are unique identifiers. They use the # to identify id's in CSS file. Once you use it once you cant use it again in the same document. You cant have two ids called main .... paste a screenshot in here. Think of an id like your drivers license number you cant have two drivers licenses with the same number. Therefore you use IDs when you want to select something specific. Classes are great classifiers. They use the . (dot) to identify the class in the CSS. They are not unique in nature. Post in screenshot here. You can also only have one id more element but you can have multiple classes per element. Post in screenshot here. Classes are used when you want to apply the same styling to multiple elements. Id gets higher priority than class in css. Meaning that any styling in css for the id will override the styling that has been given to the class. Screenshot here.
